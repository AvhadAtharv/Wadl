# Assignment 3a

Prepared for: `Atharv Avhad`

This project creates a static web server using Node.js core modules:

- `http`
- `url`
- `fs`
- `path`

## Run

```bash
npm start
```

Open `http://localhost:1800`.

The root route lists all files inside the `public` folder. Clicking a file serves its content directly.
