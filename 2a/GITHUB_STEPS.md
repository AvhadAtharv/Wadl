Use these commands after creating a GitHub repository from your own account:

```bash
git init
git config user.name "Atharv Avhad"
git config user.email "your-email@example.com"
git add .
git commit -m "Add Assignment 2a Angular application"
git branch -M main
git remote add origin https://github.com/your-username/assignment-2a.git
git push -u origin main
```
