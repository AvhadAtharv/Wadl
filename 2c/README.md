# Assignment 2c

Angular application for `Register User`, `Login User`, and `Show User Data on Profile Component`.

Prepared for: `Atharv Avhad`

## What is included

- Angular 18 project with routing and template-driven forms
- Register page that saves users in browser local storage
- Login page that validates the saved credentials
- Profile page protected with a route guard
- Navigation bar shared across the app

## Run locally

```bash
npm install
npm start
```

Open `http://localhost:4200/`.
