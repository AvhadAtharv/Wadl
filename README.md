# Wadl

# Web Development Application Lab Projects

Prepared by `Atharv Avhad`

This repository contains laboratory assignments for Web Development Application, organized by assignment code.

## Projects

### Assignment 1

- `1a` - Responsive Bootstrap admin dashboard for an online food delivery app
- `1b` - JavaScript AJAX registration flow with local storage and data listing page

### Assignment 2

- `2a` - GitHub and Git command write-up for version control setup
- `2b` - Dockerized Node.js mini app for food delivery environment setup
- `2c` - Angular user registration, login, and profile application

### Assignment 3

- `3a` - Static web server using Node.js core modules
- `3b` - CRUD API using Node.js, Express, and MongoDB with memory fallback for demo mode

### Assignment 4

- `4a` - jQuery Mobile multi-page website

## How to Run

### Static browser projects

Open these directly in a browser:

- `1a/index.html`
- `1b/index.html`
- `4a/index.html`

### Angular project

```bash
cd 2c
npm install
npm start
```

### Docker project

```bash
cd 2b
sudo docker compose up --build
```

### Static Node server

```bash
cd 3a
npm start
```

### CRUD API

```bash
cd 3b
npm install
npm start
```

If MongoDB is not running, `3b` starts in memory demo mode.
